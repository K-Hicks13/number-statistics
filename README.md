# Number Statistics

## Description

A Python console application that analyses a set of numbers entered by the user.

## Features

- Enter 10 numbers
- Display the highest value
- Display the lowest value
- Display the total
- Display the average
- Display the numbers in chronological order

## Technologies

- Python 3.13

## Future Improvements

- Support decimal numbers
- Calculate the median
- Allow the user to choose how many numbers to enter
